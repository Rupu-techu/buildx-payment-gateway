# BuildX Payment Gateway Module 💳

A modern fintech-inspired Payment Gateway frontend module built for the BuildX Challenge 2026.

This project simulates a realistic digital payment and checkout experience for online learning platforms and SaaS-style products using a responsive React architecture and modular frontend workflow.

---

# 🚀 Features

## Checkout Experience

* Responsive cart-to-checkout flow
* Modern fintech-inspired UI
* Smooth payment progression
* Premium dark-themed interface

## Dynamic Pricing System

* Real-time subtotal calculations
* Platform fee handling
* Discount calculations
* GST/tax calculation
* Coupon support

## Payment Methods

* Google Pay
* PhonePe
* Paytm
* UPI Payments
* Credit/Debit Cards
* Net Banking

## Payment UX

* Device-aware payment flow
* Desktop QR payment simulation
* Mobile payment processing simulation
* Payment validation states
* Success and failure handling

## Transaction Experience

* Animated payment success flow
* Transaction confirmation
* Downloadable PDF receipt/invoice
* Professional billing summary

# ⚙️ Installation

## Clone Repository

```bash id="j2w7vx"
git clone https://github.com/Rupu-techu/buildx-payment-gateway.git
```

## Navigate to Frontend

```bash id="m5p9qx"
cd frontend
```

## Install Dependencies

```bash id="r8w2pk"
npm install
```

## Start Development Server

```bash id="q7m4wy"
npm run dev
```

Application runs on:

```bash id="v2p8zr"
http://localhost:5173
```
